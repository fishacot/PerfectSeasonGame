# Протокол цикла аудита (82-0 / 38-0 parity)

## Роли

### Аудитор (Subagent 1)
1. Сверить механику с https://www.82-0.com/ (NBA) и https://38-0.app/ (футбол)
2. Сверить данные с Basketball Reference / FBref (выборочно 10+ игроков)
3. Прогнать `npm run build` и `npm run generate:data`
4. Записать отчёт в `docs/audit-iteration-N.md`
5. Если не 100% — написать промпт для Исправителя в `docs/fix-prompt-iteration-N.md`
6. Если 100% — записать `docs/SITE_COMPLETE.md` и остановить цикл

### Исправитель (Subagent 2)
1. Прочитать последний `docs/fix-prompt-iteration-N.md`
2. Внести правки
3. `npm run build` + `npm run generate:data`
4. Краткий отчёт о выполненных пунктах

## Критерии 100% (NBA / 82-0)

- [ ] 30 NBA команд в пуле спина
- [ ] 6+ игроков на club×era (где команда существовала)
- [ ] 5 раундов, 5 позиций PG/SG/SF/PF/C на корте
- [ ] Спин → пул только игроков этой команды и эры
- [ ] Фильтр: только игроки на открытые позиции + правила эр
- [ ] Статистика: PPG/RPG/APG/SPG/BPG (Classic), скрыта (Blind)
- [ ] 1 skip team + 1 skip era за игру
- [ ] 5 разных эр в финальном составе
- [ ] Симуляция: PTS/REB/AST/STL/BLK + category gates + non-linear curve
- [ ] Chemistry: бонус за повтор клуба (38-0 style)

## Критерии 100% (Football / 38-0)

- [ ] 11 раундов, формация 4-3-3 (минимум)
- [ ] Спин клуб + эра, пул реальных игроков
- [ ] OVR виден в Classic
- [ ] Топ-5 лиг Европы

## Критерии 100% (Hockey)

- [ ] 6 игроков, 6 разных эр
- [ ] Реальные клубы NHL
