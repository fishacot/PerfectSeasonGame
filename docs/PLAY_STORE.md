# Google Play — публикация Perfect Season

**Полный чеклист для вас и автоматизации:** [STORE_RELEASE.md](./STORE_RELEASE.md)

Команда сборки релиза: `npm run play:deploy`

---
Файл ~9.4 МБ — это **не** «тяжёлая» установка. Если крутится 10+ минут и ничего — почти всегда одно из этого:

### 1. Битый файл (Telegram)
Telegram иногда портит `.apk`. Пусть подруга проверит размер файла на телефоне:

**Должно быть ровно `9 900 015` байт (~9,44 МБ).** Если меньше — файл битый, ставить бесполезно.

**Лучше так:**
1. Залей APK на **Google Drive** / **Яндекс.Диск**
2. Она качает **с телефона** из браузера
3. Открывает через приложение **«Файлы»** / **«Мои файлы»**, не из Telegram

### 2. Play Защита сканирует и молча отменяет
- Должно появиться окно **«Установить»**
- Если есть предупреждение → **Подробнее** → **Всё равно установить**
- Xiaomi/Huawei: **Настройки → Безопасность → Установка из неизвестных источников** для «Файлы» или Chrome

### 3. Уже установилось, но не заметила
**Настройки → Приложения** → искать **«Perfect Season»**. На некоторых телефонах ярлык на рабочий стол не добавляется.

### 4. Зависший установщик
- Перезагрузить телефон
- Удалить старый скачанный APK
- Скачать заново (лучше не Telegram)
- Свободное место: минимум **500 МБ**

### Что спросить у подруги
1. Марка телефона (Samsung, Xiaomi…)
2. Размер файла в байтах/МБ
3. Было ли окно «Установить» или только «скачивание»
4. Есть ли **Perfect Season** в списке приложений

**Надёжное решение** — Google Play (внутреннее тестирование), тогда Play Защита не мешает.

---

**Отключить Play Защиту для APK, отправленного вручную, нельзя** — это защита Android для файлов «не из магазина».

| Способ установки | Play Защита |
|------------------|-------------|
| APK/AAB из Telegram, браузера, USB | Может предупредить или заблокировать → «Подробнее» → «Всё равно установить» |
| **Google Play** (официально) | Не мешает — приложение считается доверенным |

Если друг **скачал и нажал «Установить»** — игра должна появиться как **«Perfect Season»** в списке приложений. Предупреждение Play Защиты ≠ ошибка, если установка завершилась.

**Единственный нормальный способ «чтобы не мешала»** — выложить в Google Play.

## Что нужно от вас (один раз)

1. **Аккаунт разработчика** — [Google Play Console](https://play.google.com/console) (~25 USD разово).
2. **Файл для загрузки** — только **`.aab`**, не `.apk`:
   ```
   android/app/build/outputs/bundle/release/app-release.aab
   ```
3. **Политика конфиденциальности** (уже на сайте):
   - https://perfectseason.duckdns.org/ru/privacy
   - https://perfectseason.duckdns.org/en/privacy
4. **Keystore** — `android/keystore/release.keystore` **нельзя терять**. Без него нельзя обновлять приложение. Сделайте резервную копию в надёжное место (не в git).

## Сборка перед загрузкой

```powershell
npm run android:build
npm run play:preflight
```

Скрипт `play:preflight` проверит, что AAB/APK собраны и подписаны.

Перед каждым новым релизом увеличьте в `android/app/build.gradle`:
- `versionCode` — целое число (+1 каждый раз)
- `versionName` — строка для людей, например `"1.0.1"`

## Пошагово: Google Play Console

### 1. Создать приложение
- Play Console → **Создать приложение**
- Название: **Perfect Season**
- Язык по умолчанию: русский или английский
- Тип: **Игра** или **Приложение** (игра подходит лучше)
- Бесплатное

### 2. Загрузить сборку
- **Выпуск** → **Тестирование** → **Внутреннее тестирование** (быстрее всего для друзей)
- **Создать выпуск** → загрузить `app-release.aab`
- При первой загрузке включите **Play App Signing** (Google хранит ключ подписи — рекомендуется)

### 3. Контент магазина (минимум)
- **Краткое описание** (до 80 символов)
- **Полное описание**
- **Иконка** 512×512 PNG
- **Скриншоты** телефона — минимум 2 (сделайте с телефона или эмулятора)
- **Категория**: Спорт / Казуальные

### 4. Политика и безопасность
- **Политика конфиденциальности**: URL с сайта (см. выше)
- **Безопасность данных** (Data safety):
  - Classic-режим **офлайн**, личные данные **не собираем**
  - localStorage на устройстве — история игр (не уходит на сервер)
  - Вход/друзья — опционально, только если включите позже
- **Рейтинг контента** — пройти анкету (спортивная игра без насилия → обычно 3+ / Everyone)

### 5. Тестеры (для друзей до публичного релиза)
- Внутреннее тестирование → **Создать список** → добавить email Google-аккаунтов друзей
- Они получат ссылку из Play Console и ставят **из Play**, без APK и без Play Защиты

### 6. Публикация
- Сначала **Внутреннее тестирование** → проверка друзьями
- Потом **Закрытое** / **Открытое** тестирование или **Продакшен**

Проверка Google первого релиза — от нескольких часов до нескольких дней.

## APK vs AAB

| Файл | Кому |
|------|------|
| `.apk` | Ручная установка (друзья по файлу) — Play Защита может мешать |
| `.aab` | **Только Google Play** — из него Google сам делает APK для устройств |

В Play Console загружайте **только AAB**.

## Частые отказы при модерации

- Нет политики конфиденциальности → добавить URL
- Неверная анкета Data safety → указать «данные не передаются третьим лицам» для офлайн-режима
- versionCode не увеличен → +1 в `build.gradle`
- Загружен APK вместо AAB → пересобрать `npm run android:build`, взять `.aab`
